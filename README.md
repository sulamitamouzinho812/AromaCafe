# ☕ Aroma Café

Projeto desenvolvido para fins educacionais, simulando o site institucional de uma cafeteria fictícia.

O objetivo deste projeto é praticar conceitos básicos de desenvolvimento web utilizando HTML e CSS, além de introduzir o uso de Git e GitHub para trabalho colaborativo.

## 📖 Sobre o Projeto

A Aroma Café é uma cafeteria fictícia criada para servir como exemplo em atividades de programação.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Git
- GitHub

## 🚀 Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/Livia-Rachell/AromaCafe.git
```

2. Acesse a pasta do projeto:

```bash
cd aroma-cafe
```

3. Abra o arquivo `index.html` no navegador.

## 🤝 Atividade Colaborativa

Durante a atividade, cada participante será responsável por uma seção do site.

O objetivo é demonstrar como o Git permite que várias pessoas trabalhem simultaneamente em um mesmo projeto, registrando alterações e organizando o histórico de desenvolvimento.

## 📌 Observação

Este projeto possui finalidade exclusivamente educacional e foi desenvolvido para apoiar aulas introdutórias de Git, GitHub, HTML e CSS.
